# About Amazon clone

Amazon Clone is a react library for dealing with product management as well as giving you full purchase abilities

## Installation

For Installation Clone this repo and do

```
npm start
```

Once it loads youre going to have to go to the back end and start that as well you can find the back end link down below.

https://github.com/Bryanjazo/Amazon-backEnd

## Usage

For the usage of this project starting off will be a home screen filled with hundreds of products to go through. This app gives you the ability to add to basket, checkout, and purchase. You're also able to sign up and sign in in order for you to check your past purchases and have profile customization.
